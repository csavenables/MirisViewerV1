Quick Start
Get started with Miris in under 5 minutes. Choose your preferred integration method:

HTML Embed
npm Package
The simplest way to get started. Add a single script tag and start embedding 3D content:

HTML
Copy
<!DOCTYPE html>
<html>
<head>
  <!-- Load the Miris SDK from npm -->
  <script type="module" src="https://cdn.jsdelivr.net/npm/@miris-inc/components/components.mjs"></script>
  <style>
    miris-stream {
      width: 100%;
      height: 400px;
      display: block;
    }
  </style>
</head>
<body>
  <miris-stream
    key="YOUR_VIEWER_KEY"
    uuid="YOUR_ASSET_UUID"
    controls
  ></miris-stream>
</body>
</html>
üí° That's it!
Replace YOUR_VIEWER_KEY and YOUR_ASSET_UUID with your credentials.

Getting Your Credentials
Viewer Key
Your Viewer Key authenticates your application with Miris servers. Contact your Miris account representative to obtain your viewer key.

Asset UUID
Each 3D asset in your Miris library has a unique UUID. You can:

Find UUIDs in the Miris Dashboard
Fetch them programmatically using the SDK (see Fetching Assets)
<miris-stream>
Live Demo
The primary element for displaying a 3D asset. Can be used standalone or within a <miris-scene>.

Standalone Usage
When used alone, <miris-stream> creates its own rendering context:

HTML
Copy
<miris-stream
  key="YOUR_VIEWER_KEY"
  uuid="82c0d8ac-993f-44b6-b472-7b45e433b96c"
  controls
  style="width: 100%; height: 500px;"
></miris-stream>
Attributes
Attribute	Type	Required	Description
uuid	string	Yes	The UUID of the asset to display
key	string	Yes*	Your viewer key (*required when standalone)
controls	boolean	No	Enable drag-to-rotate interaction
disabled	boolean	No	Pause streaming
position	string	No	Position as "x y z" (e.g., "0 -0.5 -2")
rotation	string	No	Rotation in radians as "x y z"
scale	string	No	Scale as "x y z" (e.g., "1.5 1.5 1.5")
zoom	number	No	Uniform scale multiplier
JavaScript API
JavaScript
Copy
const stream = document.querySelector('miris-stream');

// Change asset
stream.uuid = "new-asset-uuid";

// Toggle controls
stream.controls = true;
stream.disabled = false;

// Transform properties
stream.position.set(0, -0.5, -2);
stream.rotation.y = Math.PI / 4;  // 45 degrees
stream.scale.set(2, 2, 2);
stream.zoom = 1.5;

Example: Asset Gallery
Live Demo
Create a gallery where users can select different assets:

HTML + JavaScript
Copy
<div class="gallery">
  <div class="viewer">
    <miris-scene id="scene" key="YOUR_VIEWER_KEY">
      <miris-stream id="stream" controls></miris-stream>
    </miris-scene>
  </div>
  <div class="asset-list" id="assets"></div>
</div>

<script type="module">
  await customElements.whenDefined('miris-scene');
  
  const scene = document.getElementById('scene');
  const stream = document.getElementById('stream');
  const assetList = document.getElementById('assets');
  
  // Fetch all available assets
  const assets = await scene.fetchAssets();
  
  // Create buttons for each asset
  assets.forEach(asset => {
    const btn = document.createElement('button');
    btn.textContent = asset.name;
    btn.onclick = () => {
      stream.uuid = asset.uuid;
    };
    assetList.appendChild(btn);
  });
</script>

Live Demo ‚Üí
For advanced use cases requiring full Three.js control, use the @miris-inc/three package.

Installation
Terminal
Copy
npm install @miris-inc/three three
Basic Usage
JavaScript
Copy
import { MirisScene, MirisStream } from '@miris-inc/three';
import { PerspectiveCamera, WebGLRenderer } from 'three';
import { OrbitControls } from 'three/examples/jsm/controls/OrbitControls';

// Setup
const container = document.getElementById('viewer');
const scene = new MirisScene('YOUR_VIEWER_KEY');
const camera = new PerspectiveCamera(50, container.clientWidth / container.clientHeight, 0.1, 1000);
const renderer = new WebGLRenderer({ antialias: true, alpha: true });

renderer.setSize(container.clientWidth, container.clientHeight);
container.appendChild(renderer.domElement);

// Add orbit controls
const controls = new OrbitControls(camera, renderer.domElement);

// Position camera
camera.position.set(0, 1, 5);

// Add a Miris stream
const stream = new MirisStream('YOUR_ASSET_UUID');
scene.add(stream);

// Render loop
function animate() {
  requestAnimationFrame(animate);
  controls.update();
  renderer.render(scene, camera);
}
animate();

Styling & Responsive Design
CSS Sizing
Miris elements require explicit dimensions. They won't display without width and height:

CSS
Copy
/* Block display with explicit height */
miris-stream {
  display: block;
  width: 100%;
  height: 400px;
}

/* Responsive sizing */
@media (min-width: 768px) {
  miris-stream { height: 500px; }
}

@media (min-width: 1200px) {
  miris-stream { height: 700px; }
}

Performance Tips
Lazy Loading
Only load assets when visible:

JavaScript
Copy
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    const stream = entry.target;
    stream.disabled = !entry.isIntersecting;
  });
}, { threshold: 0.1 });

document.querySelectorAll('miris-stream').forEach(stream => {
  stream.disabled = true;  // Start disabled
  observer.observe(stream);
});
